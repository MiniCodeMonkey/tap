# Carbon Theme

IBM Carbon design language for presentations

---

## Design Philosophy

The Carbon theme embodies:

- **Systematic** — Every element follows the Carbon grid
- **Industrial** — Sharp corners, no decoration
- **Functional** — Red accents mark interactive elements
- **Typographic** — IBM Plex Sans throughout

---

## Typography

<!-- pause -->

**IBM Plex Sans** for display and body text

<!-- pause -->

IBM's open-source typeface family:
- Display: Light weight 300, large scale
- Body: Regular weight 400
- Mono: IBM Plex Mono for code and data
