# Signal Theme

Developer-first, metadata-rich aesthetic

---

## Design Philosophy

The Signal theme embodies:

- **Developer Focus** — Built for technical presentations
- **Metadata Rich** — Tag and badge directives add context
- **Neon Precision** — Green accents on near-white backgrounds
- **Clean Lines** — Hairline borders replace traditional bullets

---
