## Code Blocks

```javascript
class WatercolorBrush {
  constructor(pigment) {
    this.pigment = pigment;
    this.water = 0.7; // dilution ratio
  }

  paint(canvas, stroke) {
    // Let colors flow and blend
    return canvas.apply({
      color: this.dilute(this.pigment),
      edge: 'soft',
      bleed: true
    });
  }
}
```

Code with soft edges and gentle backgrounds.
