# Mono **Theme**

Ultra-minimal weight contrast

---

## Design Philosophy

The Mono theme embodies:

- **Restraint** — Single blue accent, nothing more
- **Contrast** — Weight 300 meets weight 900
- **Typography** — The font IS the design
- **Directness** — Chevron markers, arrow elements

---

## Typography

<!-- pause -->

**Outfit** with extreme weight range (300–900)

<!-- pause -->

Weight contrast as the primary design tool:
- Display: Weight 900, uppercase
- Subtitles: Weight 300, light and airy
- Body: Weight 400, comfortable reading
- Mono: JetBrains Mono for code

---

## Code Blocks

```rust
struct Presentation {
    title: String,
    slides: Vec<Slide>,
    theme: Theme,
}

impl Presentation {
    fn new(title: &str) -> Self {
        Self {
            title: title.to_string(),
            slides: Vec::new(),
            theme: Theme::Mono,
        }
    }
}
```

Sharp 2px radius, minimal styling.
