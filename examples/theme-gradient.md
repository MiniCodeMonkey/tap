## Code Blocks

```javascript
const gradient = {
  start: '#667eea',
  middle: '#764ba2',
  end: '#f093fb',
  direction: '135deg'
};

const style = `linear-gradient(
  ${gradient.direction},
  ${gradient.start},
  ${gradient.middle},
  ${gradient.end}
)`;
```
