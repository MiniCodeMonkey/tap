# Flux Theme

Polished SaaS *product* feel

---

## Design Philosophy

The Flux theme embodies:

- **Warmth** — Off-white backgrounds, not clinical
- **Polish** — Every detail considered
- **Product** — Feels like a real app interface
- **Clarity** — Indigo accents guide the eye

---

## Typography

<!-- pause -->

**Plus Jakarta Sans** with personality

<!-- pause -->

A modern sans-serif with friendly character:
- Display: Weight 800, confident
- Body: Weight 400, approachable
- Mono: JetBrains Mono for code
- *Emphasis* gets accent color treatment
