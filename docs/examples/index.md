# Examples

Example presentations to help you get started with Tap. Each example demonstrates different features and use cases.

## Available Examples

### [Tech Talk](/examples/tech-talk)

A technical presentation template for conference talks and meetups. Features code blocks, live SQL demos, and a clean structure for explaining technical concepts.

### [Demo Day](/examples/demo-day)

A product demo template perfect for showcasing features to stakeholders. Includes sections for problem statement, solution, live demos, and next steps.

### [Workshop](/examples/workshop)

An interactive workshop template with hands-on exercises. Designed for teaching sessions with step-by-step instructions and code examples participants can follow along with.

### [Lightning Talk](/examples/lightning-talk)

A concise 5-minute talk template. Optimized for quick, impactful presentations with minimal slides and maximum focus.
